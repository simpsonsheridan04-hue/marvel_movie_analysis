This project examines the various live action Marvel comic movies that were released from 1986 to 2021. This project primiarly focuses on the different marvel film distributors and how movie budgets and profits have changed over time.

## Resources Used:
* R
  * libraries: ggplot, readr, tidyr, dplyr, stringr
  * visuals, joins, aggregates, correlations

## Data Source:

This data was collected from Kaggle (https://www.kaggle.com/datasets/minisam/marvel-movie-dataset?select=marvel_reviews.csv)
